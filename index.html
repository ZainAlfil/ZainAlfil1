HTML Basics for Kids

HTML (HyperText Markup Language) is the code used to structure content on the web. Let's explore the basics:

Headings (h1 - h6)

Headings define the structure of your page. They range from h1 (the largest) to h6 (the smallest).

HTML Grouping and Structuring Elements

HTML provides various elements to group and structure content on a webpage. Let's explore some of them:

Div (Division)

The div element is a generic container used to group other elements. It is often used for styling purposes.

This is content inside a div.

Section

The section element represents a thematic grouping of content. It is typically used to define sections in a document.

This is content inside section 1.

Header

The header element typically contains introductory content or a group of navigational links. It often includes headings and logos.

Footer

The footer element usually contains metadata, copyright information, or links to related documents.

Article

The article element represents a self-contained piece of content that could be distributed and reused independently.
